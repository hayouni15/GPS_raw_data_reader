Problem : 
Convert the GPS data provided in binary into readable data (txt or csv) file.
the GPS data is writen as follow:
         - 8 bytes: size of the header        
         - 34 bytes : header                  
         - 8 bytes : data size                
         - 48 bytes data: 
                        - time stamp      
                        - latitude        
                        - longitude       
                        - altitude        
                        - velocity        
                        - heading
